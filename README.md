# 소개
가상의 샐러드 쇼핑몰을 직접 기획하고 구현하였습니다.

# 사용 기술
- CSS
- HTML
- 바닐라 JS   
*프레임워크나 라이브러리를 사용하지 않았습니다.*

# 주요 기능

## 자동 슬라이드
![자동 슬라이드](https://user-images.githubusercontent.com/94767408/144159259-a9b256a2-52af-4525-aa19-40173801ed72.gif)
- 클릭하지 않아도 자동으로 사진을 전환합니다.
- 클릭으로도 다음 사진으로 넘길 수 있습니다.
- 마지막 사진에서 첫번째 사진으로 자연스럽게 전환합니다.

## 버튼 슬라이드
![클릭 슬라이드](https://user-images.githubusercontent.com/94767408/144160444-1eb1cd19-7048-4955-b1bb-451eab4f4ef7.gif)
- 사진을 한 장 씩 넘길 수 있습니다.
- 화면이 작아져도 한장씩 사진이 전환합니다.

## 모달창
![모달](https://user-images.githubusercontent.com/94767408/144157880-cac5d6db-c6a7-4a24-a4da-16ec7aa9b58a.gif)
- X 버튼을 누르거나 esc키를 누르면 창을 닫을 수 있습니다.

## 반응형 디자인
### 햄버거 메뉴
<큰 화면일 때 메뉴바>
![image](https://user-images.githubusercontent.com/94767408/173852058-0741c4b1-22cf-4c65-a8c8-116b3ea337f4.png)
- 전체 메뉴가 모두 나타납니다.
<작은 화면일 때 메뉴바>
![햄버거 메뉴](https://user-images.githubusercontent.com/94767408/144160818-a4a4c074-9d9f-47ef-a9c4-3c4970e0cdf4.gif)
- 메뉴 아이콘을 클릭하면 전체 메뉴를 볼 수 있습니다.
 
## 스크롤에 따라 사라지는 메뉴바
![메뉴 사라짐](https://user-images.githubusercontent.com/94767408/144161494-d51ba54c-fcb4-4971-9b7b-2496d2a8d19b.gif)
- 메인 슬라이드를 벗어나면 메뉴바가 자연스럽게 사라집니다.
